Introduction

Welcome to the documentation for the Discord CBL bot! This bot is designed to check the Community Ban List (CBL) history for a given SteamID. The bot uses GraphQL to fetch data from the Community Ban List and provides detailed information about a user's reputation, risk rating, and ban history.

Getting Started

Inviting the Bot

To invite the bot to your server, click here and follow the instructions.

Setting Up

Once the bot is in your server, make sure it has permissions to send messages and create threads in the channel it's added to.

Commands
/cbl Fetch the CBL history for a given SteamID
CBL Bot Today at 12:34 PM

Example:

/cbl steamid: 76561198064178807
